# ChalkPicks.live 🚀

ChalkPicks.live is a high-performance, AI-powered sports betting analytics platform designed for bettors who want a real edge. Built with a "Neon Cyber" aesthetic, it combines advanced LLM analysis with real-time market data from 18+ sportsbooks and prediction markets like Kalshi.

## 🌟 Key Features

### 🤖 AI-Powered Picks
- **Multi-Model Analysis**: Uses Claude for deep qualitative analysis and OpenAI for concise summaries.
- **Edge Scoring**: Every pick includes a confidence bar and calculated edge against market lines.
- **24/7 Generation**: Automated daily picks across NFL, NBA, MLB, NHL, and more.

### 📈 Professional Tools
- **+EV Finder**: Identifies positive expected value bets by comparing lines across 10+ sportsbooks.
- **Steam Move Detector**: Alerts you to sudden, sharp line movements where the "smart money" is flowing.
- **Kalshi Market Integration**: Real-time tracking of prediction markets for non-sports events and sports outcomes.
- **Arbitrage Finder**: Spot guaranteed profit opportunities through line discrepancies.

### 💎 Promotional Offers
- **5-Day Free Trial**: Full access to all premium features with no upfront cost.
- **$5 for $100 Credit**: Pay $5 and receive $100 in account balance to use towards any subscription.

### 🛡️ Edge Terminal
- **Tiered Access**: Onboarding flow that assigns users to Recreational, Serious, or Professional tiers based on their profile.
- **Advanced Metrics**: Track ROI, P&L, and Win Rate % with professional-grade dashboards.

## 🛠️ Tech Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS (Design B Neon Theme)
- **Backend**: Node.js, tRPC, Express
- **Database**: MySQL with Drizzle ORM
- **Payments**: Stripe (Subscription Tiers & Promotional Credits)
- **AI**: OpenAI (GPT-4o) & Anthropic (Claude 3.5 Sonnet)
- **Data**: The Odds API, Kalshi API, ESPN Stats API

## 🚀 Getting Started

1. **Clone the Repo**: `gh repo clone big-main/chalkpicks-prov2`
2. **Install Dependencies**: `pnpm install`
3. **Environment Setup**: Copy `.env.example` to `.env` and fill in your API keys.
4. **Database Push**: `npm run db:push`
5. **Run Dev Server**: `npm run dev`
